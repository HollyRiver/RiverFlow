심슨의 역설

여러 집단으로 나눠진 데이터들을 하나로 합쳐서 분석해버리면 어떻게 될까?

import pandas as pd
import numpy as np

2. 필요한 코드 | 비교를 위한 시각화

D. 해석

- 시각화 1 : 남자의 합격률이 더 높다 -> 성차별이 있어보인다(?)

- 시각화 2 : 학과별로 살펴보니 오히려 A, B, F, D의 경우 여성의 합격률이 높다.

- 교재에서 설명한 이유 : 여성이 합격률이 낮은 학과에만 많이 지원하였기 때문.
